CRM App With PyQt5

(DESKTOP)

Technologies

Language: Python

Library: PyQt5

Database: SQLite3

Paradigm: OOP

Architecture: MVC

This project is a desktop application built using Python and PyQt5, following the Model-View-Controller (MVC) design pattern. It leverages SQLite3 for efficient data management and employs Object-Oriented Programming (OOP) principles to ensure scalability and maintainability. The application offers a user-friendly interface and robust functionality for streamlined operations.
